BODMAS

Assignment Operators

  • = Assigns the right side of an equation to left side of assignment sign (=)

    • var number = 20;

  • += Adds the value on the right hand operand to the existing one on the left-hand operand

    • var number += 30;
    number is now = 50

  • -= Subtracts from the left-hand operand the value of the right-hand operand

    • var number -= 30;
    number is now 20

  • *= Multiplies two operands together

    • var number *= 2;
    number is now = 40

  • /= Divides the first operand by second operand

    • var number /= 4;
    number is now= 10;

  • %= Modulus Operator - Returns remainder after dividing the first operand with the second operand

    var number %= 4;
    number is now = 2

  • **= Power/exponential operator
    number **= 4;
    number is now = 16

  • + symbol can also be used for concatenating strings
